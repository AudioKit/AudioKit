#  AudioKit Internals

AudioKit has many dependencies and utilities and this is where this "internal" code is kept.
This directory is mainly for advanced users.

* Audio File
* Audio Unit Host
* Core Audio
* Exception Catcher
* EZAudio
* Microphone Tracker
* Microtonality
* Soundpipe
* Sporth
* Sporth Custom Ugens
* STK (Stanford Synthesis Toolkit)
* Synchronization
* Table
* Testing
* Utilities
